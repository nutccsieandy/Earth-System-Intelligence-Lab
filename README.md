# FlowResilienceAI Bilingual Introduction Website

A responsive bilingual (Traditional Chinese / English) one-page project website for the 2026 Presidential Hackathon International Track.

## Files

- `index.html` — Main page
- `assets/style.css` — Responsive visual design
- `assets/main.js` — Language switching, mobile menu, and animations

## Local preview

Open `index.html` directly in a browser.

For a simple local server:

```bash
python -m http.server 8080
```

Then visit:

```text
http://localhost:8080
```

## Deployment options

### GitHub Pages

1. Create or open a GitHub repository.
2. Upload all files in this folder.
3. Go to **Settings → Pages**.
4. Select **Deploy from a branch**.
5. Select `main` and `/root`.
6. Save and wait for the public URL.

### Netlify

Drag and drop this folder into Netlify Drop.

### Vercel

Import the repository into Vercel as a static website.
