PROJECT OVERVIEW

從資料碎片到可執行的災害決策

FlowResilienceAI 建立政府、救災單位與民眾之間的智慧資訊橋梁，降低資訊落差並縮短應變時間。

01
!

現行挑戰

災害資訊分散於多個機關，格式不一、更新頻率不同，難以在關鍵時刻形成完整態勢判斷。

02
✦

我們的方案

透過資料整合、智慧分析與生成式 AI，提供風險分級、事件摘要、GIS 視覺化和個人化應變建議。

03
↗

預期影響

協助決策者加速資源配置與跨機關協作，同時讓民眾獲得清楚、即時且可採取行動的安全資訊。

CORE CAPABILITIES

一個平台，串起完整災害應變流程

從即時監控、AI 分析到公眾溝通，兼顧政府管理端與民眾使用端。

⌖

即時資料整合

彙整氣象、雨量、水文、地質、土石流與災防警報等異質資料。

◫

互動式 GIS

以地圖呈現高風險區域、事件分布與環境參數，快速掌握空間態勢。

⌁

AI 風險評估

運用機器學習評估洪水、山崩與極端降雨風險，產生可解釋的分級結果。

✦

AI Copilot

透過 RAG 與 LLM 生成事件摘要、風險說明與具體應變建議。

▣

政府管理後台

支援事件管理、資源配置、跨機關協調與決策紀錄。

◉

民眾 PWA

提供行動裝置友善的警示通知、安全指引與個人化避難資訊。

SYSTEM ARCHITECTURE

可擴充、可部署、可跨區域移植

模組化架構讓不同資料來源、AI 模型與服務能彈性替換，並透過 Docker 快速部署至不同地區與情境。

  • 開放資料與即時環境資料串接
  • 機器學習、RAG 與 LLM 分析層
  • GIS、管理後台與 PWA 應用層
  • Docker 容器化與模組化部署
